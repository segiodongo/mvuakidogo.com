**"old book ui frontend"** refers to the aesthetic shown in the saved screenshot: the skills journal UI from The Long Dark — a two-page open book/journal layout rendered as an in-world object.

Key visual characteristics:
- **Layout**: Two-page open book spread — left page is a navigable list, right page is a detail/content view
- **Material**: Aged, yellowed parchment/paper texture with worn edges, staining, and foxing — feels physically old and used
- **Color palette**: Sepia/tan/brown paper tones, dark brown/black ink for text and icons, muted teal/blue-grey for highlighted/selected rows and progress fills, desaturated overall
- **Typography**: All-caps spaced serif or slab-serif for headings ("SKILLS", "COOKING"), small caps for labels, body text in a readable serif — printed/stamped feel, not digital
- **Icons**: Small monochrome hand-drawn or engraved-style pictograms (knife, flame, fishing hook, rifle, bow, spool) in square icon frames with a diamond badge showing level number
- **List items**: Left-aligned rows with icon + level badge + label + horizontal progress bar; selected row has a dark highlight band
- **Detail panel (right page)**: Sepia-toned illustration/photo at top, level indicator with diamond shape, teal highlight rows for stat bonuses in small caps, body description text in serif below a ruled line
- **Progress bars**: Thin horizontal bars with dark fill on parchment background
- **Navigation**: Tab icons at top of left page (category switcher), "BACK" button in bottom-right corner in dark panel
- **Overall feel**: Diegetic UI — the interface IS the journal. Handcrafted, analogue, survivalist. No chrome, no glass, no glow — just ink on paper.

When the user says "old book ui frontend" they want UI to evoke this — open book layouts, parchment textures, engraved icons, sepia tones, list+detail two-panel spreads, and the sense that the interface is a physical object.
