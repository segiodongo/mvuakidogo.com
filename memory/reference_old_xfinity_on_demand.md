**"old xfinity on demand"** refers to the aesthetic of the mid-2000s Comcast / Xfinity cable box on-screen menu (DVR Menu / On Demand / Find Shows screens).

Key visual characteristics:
- **Background**: Strong blue gradient — brighter/lighter blue at the top, deeper navy toward the bottom. Flat, slightly glossy cable-box look.
- **Header bar**: Top strip with the provider wordmark on the left (Xfinity / Comcast), a clock in the middle-ish (e.g. "8:11pm"), and a red "TV GUIDE" badge in the top-right corner.
- **Top-right preview window**: A live/preview video thumbnail box in the upper-right showing the currently highlighted item (with a small caption label over it).
- **Top-left info area**: Title + a short description of the currently selected item, with a small icon to its left.
- **Menu options**: Rounded **pill/oval buttons** — blue fill with thin white/light borders, white centered text. Arranged in **two columns**.
- **Selection highlight**: The currently focused option is lit in a **pale yellow** (cream/butter yellow) pill with dark text.
- **Typography**: Clean bold sans-serif, white text on blue, drop shadows for legibility.
- **Footer**: Sometimes a "More ▼ Choices" affordance at the bottom center.
- **Overall feel**: Glossy early-digital-cable TV menu — blue gradient, rounded pills, yellow selection, corner preview video, TV-Guide branding.

When the user says "old xfinity on demand" they want UI to evoke this exact cable-box menu language — blue gradient, rounded oval pill rows in two columns, pale-yellow selection, top-right preview thumbnail, top-left description of the selected item.
